Introduction

NodeJS is an single-threaded event driven Javascript runtime. NodeJS can allow you to run server-side Javascript with non-blocking I/O operations for scalable web applications. NodeJS also allows for child processes to utilize multiple CPU cores of a system. NodeJS is supported by Node Package Manager (npm) and includes hunders of thousands of packages for reusable code.

  • Node.js is an open source server environment
  • Node.js is free
  • Node.js runs on various platforms (Windows, Linux, Unix, Mac OS X, etc.)
  • Node.js uses JavaScript on the server
  • Node.js is built on Google Chromes V8 engine

Getting Started

After installing NodeJS, create a Javascript file called "app.js" and insert the follwing code or your own Javascript.

//app.js
function myFunction(string){
    console.log(string)    
}

myFunction(`I'm Started!`)

After creating the file, excute through a terminal with 

node app.js

you should see "I'm Started!" in the console. Now your executing Javacsript through NodeJS!

To start a simple HTTP server copy the following code into a new file and run using NodeJS through a terminal.


  const http = require('http');

  const hostname = '127.0.0.1';
  const port = 3000;

  const server = http.createServer((req, res) => {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Hello World\n');
  });

  server.listen(port, hostname, () => {
    console.log(`Server running at http://${hostname}:${port}/`);
  });

To run the app type "node app.js" in a terminal and visit http://localhost:3000 in a web browser.

Modules

NodeJS uses modules to import javascript files into your application.

You can create your own modules by using module.exports

//addition.js
module.exports.addTwoNumbers = function(first_number, second_number){
 return first_number + second_number
}

To include the new module just use require()

var add = require('addition')
console.log(add.addTwoNumbers(2, 4)) // should see 6

You can also use other libraries in the same manner.

NPM

NPM is a package manager for Node.js packages, or modules if you like. www.npmjs.com hosts thousands of free packages to download and use. The NPM program is installed on your computer when you install Node.js

Open the command line interface and tell NPM to download the package you want. To download a package called "upper-case" use the following command:

C:\Users\Your Name>npm install upper-case

To use the module in you application use the require() function as shown in the modules sections

Debugging

Debugging can be achieved by a few ways in NodeJS. Some options for debugging include:

  • node-inspect CLI Debugger supported by the Node.js Foundation which uses the Inspector Protocol. A version is bundled with Node and can be used with 
    node --inspect myscript.js.
  • Chrome DevTools 55+ Open chrome://inspect in a Chromium-based browser. Click the Configure button and ensure your target host and port are listed.
  • Visual Studio Code 1.10+ In the Debug panel, click the settings icon to open .vscode/launch.json. Select "Node.js" for initial setup.
